About BharatGyan

BharatGyan is an AI-powered personalized learning ecosystem connecting students, teachers, and parents. It uses gamification, personalized feedback, and multilingual support to improve learning outcomes and boost engagement.

Developed by TechNari Shakti, this project is part of Smart India Hackathon 2025.

Note: This is just a demo prototype showcasing the frontend user interface and design concept only. The full functional backend and AI-powered system are under development.

Note: Visual elements like progress bars in this prototype are simplified representations and can be implemented as advanced charts, graphs, and interactive visuals in the final product.
